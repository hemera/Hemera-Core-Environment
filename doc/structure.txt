apps, bin, config, log

apps: Contains all the applications.
APPLICATION folders using the application names

apps/APPLICATION: Contains all the modules.
MODULE folders using the module names, lib, ham file

apps/APPLICATION/MODULE: Contains module classes Jar file and optional configuration file.
MODULE.jar with the same name as the module, config.file

apps/APPLICATION/lib: Contains all the library files used by all modules.

apps/APPLICATION/ham: Application model file.

bin: Contains all the internal libraries and hemera script file.

config: Contains hemera.env runtime environment configuration file.

log: Contains all log files for all applications.
